# Svault Logo Design Prompt

## English Version

Create a minimalist, modern logo for "Svault" - a content-addressed media archive system built with Rust.

### Visual Concept

**Core Element**: A geometric vault door constructed from hexagonal tiles, representing hash-based content addressing.

**Composition**:
- Central hexagonal honeycomb pattern (5-7 hexagons)
- Each hexagon contains subtle gradients suggesting photo/video content
- The pattern flows into a pipeline/stream on the right side (3-4 connected segments)
- Optional: Small padlock icon integrated into one hexagon

**Color Palette**:
- Primary: Deep slate blue (#2C3E50) for vault structure
- Accents:
  - Coral red (#FF6B6B) - warmth, memories
  - Amber yellow (#FFD93D) - light, photography
  - Cyan green (#6BCF7F) - freshness, integrity
- Background: Transparent or white

**Style**:
- Flat design with subtle depth (2-3px shadows)
- Clean geometric lines, 2px stroke weight
- Technical but approachable aesthetic
- Vector-style, scalable

**Typography** (if included):
- Modern sans-serif (e.g., Inter, Manrope, or similar)
- The "S" in "Svault" could incorporate a hexagon shape
- Weight: Medium to Bold

**Layout Options**:
1. Icon only (hexagon vault with pipeline)
2. Icon + wordmark (horizontal)
3. Icon + wordmark (stacked)

### Design Principles

- **Trustworthy**: Solid geometric forms convey reliability
- **Efficient**: Pipeline flow suggests streamlined processing
- **Modern**: Clean lines and contemporary color palette
- **Technical elegance**: Sophisticated but not intimidating

### Inspiration References

- Rust logo's geometric precision and interlocking gears
- Git's branching structure and node connections
- Vault architecture (bank vaults, safe deposit boxes)
- Honeycomb patterns in nature (efficiency, structure)

### Technical Requirements

- Format: SVG (vector)
- Size: 512x512px artboard minimum
- Works on both light and dark backgrounds
- Monochrome version should remain recognizable
- Scalable from 16x16px (favicon) to large prints

### Mood Board Keywords

`content-addressed storage` `cryptographic hashing` `media archive` `rust programming` `pipeline architecture` `data integrity` `hexagonal grid` `vault security` `photo preservation` `unix philosophy`

## 中文版

为 "Svault" 设计一个极简现代的 Logo - 这是一个用 Rust 构建的基于内容寻址的媒体归档系统。

### 视觉概念

**核心元素**：由六边形瓦片构成的几何保险库门，代表基于哈希的内容寻址。

**构图**：
- 中心六边形蜂窝图案（5-7个六边形）
- 每个六边形内部有微妙的渐变，暗示照片/视频内容
- 图案右侧流动成管道/流的形状（3-4个连接段）
- 可选：在某个六边形中集成小挂锁图标

**配色方案**：
- 主色：深石板蓝 (#2C3E50) 用于保险库结构
- 强调色：
  - 珊瑚红 (#FF6B6B) - 温暖、回忆
  - 琥珀黄 (#FFD93D) - 光线、摄影
  - 青绿色 (#6BCF7F) - 清新、完整性
- 背景：透明或白色

**风格**：
- 扁平设计带微妙深度（2-3px 阴影）
- 简洁的几何线条，2px 描边粗细
- 技术感但平易近人的美学
- 矢量风格，可缩放

**字体**（如果包含）：
- 现代无衬线体（如 Inter、Manrope 或类似字体）
- "Svault" 中的 "S" 可以融入六边形形状
- 字重：中等到粗体

**布局选项**：
1. 仅图标（六边形保险库 + 管道）
2. 图标 + 文字标识（横向）
3. 图标 + 文字标识（堆叠）

### 设计原则

- **可信赖**：坚实的几何形式传达可靠性
- **高效**：管道流动暗示流线型处理
- **现代**：简洁的线条和当代配色
- **技术优雅**：精致但不令人生畏

### 灵感参考

- Rust logo 的几何精确性和互锁齿轮
- Git 的分支结构和节点连接
- 保险库建筑（银行金库、保险箱）
- 自然界的蜂窝图案（效率、结构）

### 技术要求

- 格式：SVG（矢量）
- 尺寸：最小 512x512px 画板
- 适用于浅色和深色背景
- 单色版本应保持可识别性
- 可从 16x16px（favicon）缩放到大型印刷品

### 情绪板关键词

`内容寻址存储` `加密哈希` `媒体归档` `Rust 编程` `管道架构` `数据完整性` `六边形网格` `保险库安全` `照片保存` `Unix 哲学`

## Design Iterations

After generating the initial design, consider these variations:

1. **Simplified version**: Reduce to 3-5 hexagons for better scalability
2. **Animated version**: Hexagons filling with color in sequence (for web)
3. **Monochrome version**: Single color for print/embroidery
4. **Favicon version**: Ultra-simplified icon for 16x16px display
5. **Dark mode version**: Adjust colors for dark backgrounds
